# 🛵 Fast Deliveries - Logic Puzzle in Prolog

## 📖 Description - Activity 1

This project is a **Prolog** implementation of the *"Fast Deliveries"* logic puzzle, a challenge inspired by the Racha Cuca style. The objective is to deduce the characteristics of five delivery riders 🏍️ and their deliveries 📦 across different regions of the city 🏙️ using a set of provided clues.

The exercise was developed for the **Logic Programming** course, applying concepts of **logical reasoning**, **facts**, and **rules** for automated inference 🤓.

## ⚡ Features

- Representation of **facts and rules** in Prolog 📜.
- Deduction of the characteristics of each delivery rider 🧑‍🦰🧑‍🦱🧑‍🦳.
- Determination of delivery regions based on the clues 🗺️.
- Example of solving a complex logic puzzle using Prolog 🧩.

## 🛠️ Requirements

- [SWI-Prolog](https://www.swi-prolog.org/) installed to run the code 💻.
- An operating system compatible with SWI-Prolog 🖥️.
